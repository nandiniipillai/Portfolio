# Nandini Pillai — Portfolio

Product & experience designer portfolio, built with Next.js 15 (App Router, JavaScript) and Tailwind CSS.

## Local dev

```bash
npm install
npm run dev   # http://localhost:3000
npm run build # production build
npm run lint
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Go to https://vercel.com/new → Import Git Repository → pick this repo.
3. Framework Preset: Next.js. Root Directory: `./` (leave defaults).
4. No environment variables required.
5. Click Deploy. First deploy takes ~90s.
